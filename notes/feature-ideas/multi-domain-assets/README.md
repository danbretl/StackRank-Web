# Multi-domain visual references

`books-primary-screen-concept.png` is an AI-generated visual direction created July 13, 2026 from
the current StackRank Movies desktop screen. It is a design reference, not a shipped screenshot or
provider artwork source.

The implemented `/books` slice preserves the monochrome family, centered Rank / Ranking / You
navigation, editorial search field, and three four-cover discovery shelves. It intentionally shows
an honest empty-user state and device-only/provider disclosure instead of the concept's populated
recent-ranking rail.

Rendered implementation screenshots are generated by the focused browser flow under
`reports/e2e/latest/screenshots/` (gitignored).
